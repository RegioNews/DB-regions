# GitHub Pages deployment

## Что внутри
- `index.html` — главный файл сайта для GitHub Pages

## Как опубликовать
1. Создайте новый репозиторий на GitHub.
2. Загрузите в него содержимое этой папки.
3. Откройте `Settings` → `Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/ (root)`.
6. Сохраните настройки.
7. Сайт появится по адресу вида `https://USERNAME.github.io/REPOSITORY/`.

## Важно
- Главная страница уже названа `index.html`, это подходит для GitHub Pages.
- Для live-RSS открывайте сайт по HTTP/HTTPS, а не через `file://`.
